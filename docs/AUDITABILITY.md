# Auditability & Verification

SPECTRUM GSE is designed for **third-party audit**.

## What Can Be Audited
- Input distributions
- Output metrics
- Refusal events
- Invariant enforcement
- CI logs

## What Cannot Be Hidden
- Scalar refusal logic
- Topology mutation checks
- Learning prohibition
- Determinism

## Intended Auditors
- Academic reviewers
- Internal research oversight
- Enterprise validation teams
- Independent verifiers
